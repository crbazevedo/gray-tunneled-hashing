# Experiments - Real Data

Este diretório contém experimentos e resultados com dados reais e sintéticos.

## Estrutura

- `reports/`: Relatórios detalhados e sumários
  - `SPRINT8_BENCHMARK_RESULTS_REPORT.md`: Relatório completo do benchmark Sprint 8
  - `SPRINT8_BENCHMARK_ANALYSIS.md`: Análise automática do benchmark
  - `RECALL_RESULTS_SUMMARY.md`: Sumário histórico de resultados de recall
  - `STATUS_ANALYSIS.md`: Análise de status geral do projeto
- `results_json/`: Resultados experimentais em formato JSON (não versionados)
- `data/`: Datasets (sintéticos e reais)

## Relatórios Principais

### Sprint 8 Benchmark
- **SPRINT8_BENCHMARK_RESULTS_REPORT.md**: Relatório completo com análise detalhada
- **SPRINT8_BENCHMARK_ANALYSIS.md**: Tabelas comparativas e métricas
- **SPRINT8_BENCHMARK_SUMMARY.md**: Resumo executivo

### Histórico
- **RECALL_RESULTS_SUMMARY.md**: Evolução dos resultados de recall ao longo dos sprints
- **STATUS_ANALYSIS.md**: Status geral e análise do projeto

## Resultados Experimentais

Os arquivos JSON com resultados detalhados estão em `results_json/` e não são versionados para manter o repositório limpo. Para acessar resultados específicos, consulte os relatórios em `reports/`.
